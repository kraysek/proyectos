# Proyectos Personales y Ejemplos Técnicos

Este repositorio contiene scripts y proyectos desarrollados en mi tiempo libre para aprender y practicar habilidades técnicas.

⚠️ Nota: Ningún archivo contiene información confidencial de mi empleador actual. Los ejemplos son genéricos y educativos.

## Contenido destacado
- `VeRespaldo.sh`: Validación automática de respaldos con SSH y checksum (ejemplo educativo)
- `backup.sh`: Automatización básica de copias de seguridad
- `portscaner.sh`: Escaneo de puertos local (uso personal)
- Proyecto QR para niños vulnerables: Iniciativa social sin relación con mi trabajo profesional
